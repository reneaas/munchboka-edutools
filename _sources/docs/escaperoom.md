# `escaperoom` directive

`escaperoom` is an alias for [`escape-room`](escape-room.md).

Creates a sequential puzzle flow where each room is unlocked by a code.

## Example

````markdown
:::{escaperoom}
Puzzle: Room 1
Code: 42
Q: What is $20+22$?

Puzzle: Room 2
Code: 144
Q: What is $12\times 12$?
:::
````

Use the canonical page for full details: [`escape-room`](escape-room.md).
