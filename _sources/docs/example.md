# `example` directive

Creates a styled worked-example admonition.

## Signature

- Required arguments: `1`
- Optional arguments: `0`
- Body content: `yes`

## Options

_No options._

## Example

````markdown
:::{example} Solve a quadratic
Factor $x^2-5x+6=(x-2)(x-3)$.
:::
````
