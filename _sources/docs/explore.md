# `explore` directive

Creates an exploratory activity admonition.

## Signature

- Required arguments: `1`
- Optional arguments: `0`
- Body content: `yes`

## Options

_No options._

## Example

````markdown
:::{explore} Investigate
Try several values of $x$ and describe the pattern.
:::
````
