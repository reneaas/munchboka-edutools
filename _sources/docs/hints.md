# `hints` directive

Creates a hints admonition (collapsed by default).

## Signature

- Required arguments: `0`
- Optional arguments: `1`
- Body content: `yes`

## Options

| Option | Type |
|---|---|
| `dropdown` | string |

## Example

````markdown
:::{hints}
Start by factoring out the common term.
:::
````
