# `sign-chart` directive

`sign-chart` is an alias for [`signchart`](signchart.md).

Generates a sign chart for a function (original implementation).

## Example

````markdown
```{sign-chart}
---
function: x**2 - 4, f(x)
factors: true
width: 80%
---
```
````

Use the canonical page for full details: [`signchart`](signchart.md).
