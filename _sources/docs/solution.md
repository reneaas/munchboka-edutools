# `solution` directive

Creates a full-solution admonition (collapsed by default).

## Signature

- Required arguments: `0`
- Optional arguments: `1`
- Body content: `yes`

## Options

| Option | Type |
|---|---|
| `dropdown` | string |

## Example

````markdown
:::{solution}
Step 1: Set up the equation.

Step 2: Solve for $x$.
:::
````
