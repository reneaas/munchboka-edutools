# `summary` directive

Creates a chapter/section summary admonition.

## Signature

- Required arguments: `1`
- Optional arguments: `0`
- Body content: `yes`

## Options

_No options._

## Example

````markdown
:::{summary} Key points
- Derivative = rate of change
- Integral = accumulation
:::
````
