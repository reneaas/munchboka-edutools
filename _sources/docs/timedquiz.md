# `timedquiz` directive

`timedquiz` is an alias for [`timed-quiz`](timed-quiz.md).

Creates a timed multiple-choice quiz with countdown and scoring.

## Example

````markdown
:::{timedquiz}
Q: What is $5-2$?
+ 3
- 2
- 4

Q: What is $2^3$?
+ 8
- 6
- 9
:::
````

Use the canonical page for full details: [`timed-quiz`](timed-quiz.md).
