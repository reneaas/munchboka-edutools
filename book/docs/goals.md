# `goals` directive

Creates a learning-goals admonition.

## Signature

- Required arguments: `1`
- Optional arguments: `0`
- Body content: `yes`

## Options

_No options._

## Example

````markdown
:::{goals} Learning goals
- Explain slope
- Compute simple derivatives
:::
````
