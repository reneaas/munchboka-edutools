# `pairpuzzle` directive

`pairpuzzle` is an alias for [`pair-puzzle`](pair-puzzle.md).

Creates a drag-and-drop matching puzzle from `left : right` pairs.

## Example

````markdown
:::{pairpuzzle}
$2+2$ : 4
$3^2$ : 9
$\sin^2(x)+\cos^2(x)$ : 1
:::
````

Use the canonical page for full details: [`pair-puzzle`](pair-puzzle.md).
