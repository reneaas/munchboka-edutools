# `signchart2` directive

`signchart2` is an alias for [`signchart-2`](signchart-2.md).

Generates an enhanced sign chart with broader function support.

## Example

````markdown
```{signchart2}
---
function: sin(x)*cos(x), f(x)
domain: (-3.5, 3.5)
factors: true
width: 100%
---
```
````

Use the canonical page for full details: [`signchart-2`](signchart-2.md).
